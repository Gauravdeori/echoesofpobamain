# Echoes of Poba

A community-driven environmental initiative working to clean, protect, and preserve **Poba Reserve Forest** in Dhemaji district, Assam, along the Assam–Arunachal Pradesh border.

## 🌿 Project Overview

Echoes of Poba is designed to promote nature conservation, community awareness, and youth empowerment. The platform features:
- **World Environment Day Portal**: Detailed event itineraries, live countdowns, and fully integrated volunteer registration systems.
- **Interactive Photo Gallery**: Authentic visual showcases of cleanups, conservation walks, and local species.
- **Volunteer & Newsletter Integrations**: Secure, AJAX-based contact forms sending applications directly to the core inbox.
- **Premium Nature Theme**: Smooth glassmorphic statistics overlays, cohesive organic color palettes, and fully fluid responsive grids across mobile, tablet, and desktop viewports.

---

## 🛠️ Tech Stack

- **Core**: React 18, TypeScript, Vite
- **Styling**: Tailwind CSS, Lucide icons, Tailwind Animate
- **Components**: shadcn-ui primitives (Radix UI)
- **Forms**: AJAX-based FormSubmit integration
- **Testing**: Vitest, React Testing Library, JSDOM

## 🚀 Getting Started

### Prerequisites

Make sure you have Node.js installed (v18+ recommended) along with a package manager (`npm` or `bun`).

### Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/Gauravdeori/echoesofpobamain.git
   cd echoesofpobamain
   ```

2. **Install dependencies**:
   ```bash
   npm install
   # or
   bun install
   ```

3. **Start the local development server**:
   ```bash
   npm run dev
   # or
   bun dev
   ```
   Open [http://localhost:8080](http://localhost:8080) to view it in the browser.
